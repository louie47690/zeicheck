## ℹ️ What is zeicheck?

zeicheck is a simple tool to check your e-Tax (xtx) tax return files for errors. It ensures your tax data is consistent before submission. This reduces the chance of mistakes that might cause problems with your tax filing.

zeicheck works with Japan’s e-Tax format called "xtx." It reviews your tax return data and points out areas that might need fixing. By using zeicheck, you save time and avoid frustration during your tax process.

## ▶️ How to Use zeicheck

Using zeicheck is easy. The tool runs in a command window or terminal but requires no programming skills.

### Steps to Check Your Tax Data

1. **Open the Application:**
   - Windows: Find zeicheck in your Start menu.
   - macOS: Open Applications and click zeicheck.
   - Linux: Open your terminal and type `zeicheck` then hit Enter.

2. **Prepare Your xtx File:**
   - Locate your e-Tax file, usually with the `.xtx` extension.
   - Save it in a folder you can easily find.

3. **Run the Check:**
   - When zeicheck opens, it will ask you to enter the path of your xtx file.
   - You can type or paste the full path or drag the file into the window if your system supports it.
   - Press Enter or click the check button.

4. **Review the Results:**
   - zeicheck shows messages about data consistency.
   - It will list any errors or warnings found.
   - Errors are problems that must be fixed.
   - Warnings suggest things you should review but may not stop your filing.

5. **Fix Issues:**
   - Use your usual e-Tax software to correct issues in the xtx file.
   - Run zeicheck again until it shows no errors.
